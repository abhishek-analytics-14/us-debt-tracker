## US Debt Tracker Project

Overview

This project analyzes historical U.S. national debt data using Excel. It includes data cleaning, trend analysis, visual dashboards, and key insights to understand how the debt has evolved over time.


 Tools Used
 
- Microsoft Excel
- PivotTables
- Charts & Dashboards
- Data Cleaning Techniques


Project Files

US Debt Tracker Project.xlsx (Full Excel workbook with cleaned data, analysis, and dashboard)

Key Insights

- Debt growth trends over multiple decades
- Year‑over‑year changes
- Visual dashboard summarizing major patterns
- Identification of high‑growth periods


Skills Demonstrated

- Data cleaning
- Exploratory data analysis
- Dashboard creation
- Business storytelling
- Excel automation techniques
